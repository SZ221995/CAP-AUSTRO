Capitan Austro - paquete para GitHub Pages

Contenido:
- index.html  -> archivo principal del sitio
- .nojekyll   -> evita procesamiento innecesario de Jekyll

Pasos rápidos:
1) Crear un repositorio público en GitHub.
2) Subir estos archivos al repositorio.
3) Ir a Settings > Pages.
4) En Build and deployment, elegir 'Deploy from a branch'.
5) Branch: main ; Folder: /(root).
6) Guardar y esperar la URL pública https://usuario.github.io/repositorio/.
7) Abrir Genially > Insert > Others > pegar el código iframe apuntando a la URL pública.

Iframe sugerido:
<iframe src="https://TU-USUARIO.github.io/TU-REPOSITORIO/" width="100%" height="720" frameborder="0" allowfullscreen></iframe>
